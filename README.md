# Nikel

Este é um repositório de exemplo para o projeto [Codaí](https://plataforma.growdev.com.br/curso/codai).

This is an example repository for the [Codaí Project](https://plataforma.growdev.com.br/curso/codai).

Pude aprender o básico de HTML com esse projeto, onde desenvolvi uma página em que é possível armazenar as entradas e saídas de dinheiro, para ter um controle financeiro. A página contém login com usuário e senha, sendo possível criar um usuário novo, e a página é capaz de armazenar os dados (seja no armazenamento local ou num servidor onde a página pode ser hospedada).
